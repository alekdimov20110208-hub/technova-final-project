Финален проект по HTML и CSS
Тема: TechNova Studio - професионален сайт за IT/уеб агенция

Файлова структура:
- index.html
- za-nas.html
- uslugi.html
- kontakti.html
- thanks.html
- style.css
- img/
  - home.svg
  - team.svg
  - web-design.svg
  - coding.svg
  - support.svg
  - contact.svg

Покрити изисквания:
1. Общо 4 HTML страници.
2. Всички страници са свързани с меню.
3. Има заглавия, параграфи, изображения, линкове, хоризонтална линия, нов ред, списъци, bold/strong/italic текст.
4. Всички изображения са в папка img.
5. На всяка страница има поне 2 структурни елемента.
6. Използвани са background, color, width, margin, padding, border, border-radius, display, text-align и box-shadow.
7. На страницата uslugi.html има минимум 2 карти, разположени една до друга чрез CSS grid.

Подобрения:
- Добавен е viewport meta tag за по-добро мобилно оформление.
- Използвани са semantic HTML елементи: header, nav, main, section, article и footer.
- Текстът е коригиран и запазен като UTF-8.
- Футърът е направен по-професионален.
- Контактната форма пренасочва към страница за успешно изпращане.
